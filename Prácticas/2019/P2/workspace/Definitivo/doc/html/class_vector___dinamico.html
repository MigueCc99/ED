T.D.A Vector_Dinamico.

Una instancia vector_dinamico del tipo de datos abstracto Vector_Dinamico es un objeto que representa un conjunto de elementos. A su vez, esta instancia Vector_Dinamico se compone de un puntero a T, un numElementos y un reservados
