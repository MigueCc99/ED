Descripción detallada

T.D.A Ingrediente.

Una instancia ingr del tipo de datos abstracto Ingrediente es un objeto que representa un ingrediente. A su vez, este ingrediente se compone de dos cadenas de caracteres, un nombre y un tipo de ingrediente. Además dispone enteros que determinan la cantidad de calorias, hidratos, proteinas, grasas y fibras que contiene el ingrediente.

Lo representamos como

nombre, calorías, hidratos, proteinas, grasas, fibras, tipo.
